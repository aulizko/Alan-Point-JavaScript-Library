Class SpinnerOnSteroids

Single spinner object. Ideal if spinner alone on page, or spinners cannot be unified into one container

Constructor

SpinnerOnSteroids ( html , callback , context , step , min , max )
Parameters:
html <Object} object with properties:
  • container {jQuery} jQuery reference to html element (container)
  • input {jQuery} jQuery reference to input element
  • up {jQuery} jQuery reference to increase control
  • down {jQuery> jQuery reference to decrease control
callback <Function> function which invocated every time when spinner value changed. Spinner object as it is will be passed to callback function as first parameter.
context <Object> object which will be passed to callback function as this element.
step <Number> by this number input value will be changed every time user clicks on control
min <Number> min value of input. Default to 0.
max <Number> max value of input. Default to 255.
