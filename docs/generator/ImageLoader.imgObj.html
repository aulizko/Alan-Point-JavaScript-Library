ImageLoader.imgObj ( el , url )
Parameters:
el <jQuery> HTML DOM id of the image element

Properties

_fetched - private {Boolean}

Whether the image has already been fetched. In the case of a foldCondional group, keeps track for when the trigger is fired so images aren't fetched twice

el - {jQuery}

jQuery reference to the HTML image element

height - {Number}

Pixel height of the image. Will be set as a "height" attribute after the image is fetched. Detaults to the natural height of the image.

setVisible - {Boolean}

Whether the style.visibility should be set to "visible" after the image is fetched. Used when setting src images as visibility:hidden prior to image fetching

url - String

URL for the image

width - {Number}

Pixel width of the image. Will be set as a "width" attribute after the image is fetched. Detaults to the natural width of the image.
