log

AP log ( msg , cat , src )
If the 'debug' config is true, a 'ap:log' event will be dispatched, which the logger widget and anything else can consume. If the 'useConsole' config is true, it will write to the browser console if available.
Parameters:
msg <String> The message to log.
cat <String> The log category for the message. Default categories are "info", "warn", "error", time". Custom categories can be used as well. (opt)
src <String> The source of the the message (opt)
Returns: AP
